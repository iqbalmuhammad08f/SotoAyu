# Soto Ayu

Sebuah aplikasi dekstop untuk warung makan Soto Ayu yang terdapat dua role yakni Admin dan Kasir, serta di setiap rolenya memiliki menu fitur tersendiri-sendiri.

## Fitur
### Admin
- laporan keuangan
- CRUD akun kasir
- CRUD karyawan
- CRUD menu
- History transaksi
### Kasir
- transaksi
- History transaksi
### Teknologi yang Digunakan
- C#
- .Net Framework
- Winfrom
